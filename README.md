# ChatGPT_Interest_TimeSeries
|**Español** | **English** | 
|-------------------|----------------|
|Este código en R es una serie de pasos para modelar el interés en el tiempo en ChatGPT usando datos de GoogleTrends.|This code in R is a series of steps for modelling interest over time in ChatGPT using data from Google Trends.|
|To arrive at the final model, different approaches were tested, including ARMA, ARIMA, ETS, Fourier, and seasonal decompositions, in which weekly and annual patterns were discovered. Interest declined on Saturdays and Sundays and annually during the summer and winter holiday periods. |Para llegar al modelo final, se probaron diferentes enfoques, incluyendo ARMA, ARIMA, ETS, Fourier y descomposiciones estacionales, en los que se descubrieron patrones semanales y anuales. El interés disminuyó los sábados y domingos y, anualmente, durante los periodos vacacionales de verano e invierno. |
